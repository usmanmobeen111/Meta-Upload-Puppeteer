This is a placeholder file for the application icon.

To complete the Windows installer setup, you need to:

1. Create or obtain a 256x256px logo for your application
2. Convert it to .ico format using a tool like:
   - https://www.icoconverter.com/
   - https://convertio.co/png-ico/
   - Or use ImageMagick: convert logo.png -define icon:auto-resize=256,128,64,48,32,16 icon.ico

3. Replace this file with your actual icon.ico

The icon will be used for:
- Windows installer
- Desktop shortcut
- Taskbar
- App window title bar

Recommended sizes in the .ico file: 16x16, 32x32, 48x48, 64x64, 128x128, 256x256
